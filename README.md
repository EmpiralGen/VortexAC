VortexAC is a high-performance Minecraft anti-cheat built for competitive servers that value fairness, stability, and accuracy.  
It is designed to detect and prevent unfair gameplay while keeping false positives low and server performance stable.

VortexAC is available in both Free and Paid editions.

--------------------------------------------------

About

VortexAC focuses on clean detection logic, optimized performance, and reliable flagging.  
It is developed for PvP and competitive environments where precision matters.

The goal is simple:
Accurate detection. Minimal server impact. Consistent results.

--------------------------------------------------

Features

Combat Checks
- KillAura detection
- Reach detection
- AutoClicker detection
- Velocity consistency checks

Movement Checks
- Speed detection
- Fly detection
- Invalid motion checks
- NoSlow detection

Exploit Protection
- Packet validation
- Timer checks
- Basic crash protection

Core System
- Low false positive design
- Lightweight architecture
- Clean violation tracking
- Configurable punishments

--------------------------------------------------

Free vs Paid

Free Edition:
- Core combat and movement checks
- Basic configuration
- Standard updates

Paid Edition:
- Advanced predictive checks
- Improved reach & velocity system
- More detailed logging
- Advanced punishments & bypass protection
- Priority updates and support

--------------------------------------------------

Performance

VortexAC is optimized to run efficiently without impacting TPS.
Checks are designed to avoid unnecessary processing and reduce memory usage.

--------------------------------------------------

Installation

1. Download the latest VortexAC release.
2. Place the .jar file in your server's plugins folder.
3. Restart the server.
4. Configure settings in config.yml.
